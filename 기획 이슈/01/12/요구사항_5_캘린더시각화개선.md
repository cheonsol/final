
## 🎯 목표

### 사용자 스토리
관리자로서, 캘린더에서 근태 유형을 색상으로 쉽게 구분하고 싶습니다.

### 핵심 목표
- 근태 유형별 색상 구분
- 상태별 투명도 적용
- 휴재 일수 명확한 표시

---
## 📋 상세 요구사항

### 5-1. 색상 구분 강화

#### 현재 상황
- 선으로만 표시

#### 개선 사항 - 근태 유형별 색상 구분
| 근태 유형 | 색상 |
| --- | --- |
| 연차 | 파란색 |
| 휴재 | 회색/패턴 |
| 워케이션 | 보라색/그라데이션 |
| 일반 근무 | 초록색 |
| 야근 | 주황색 |

---

### 5-2. 상태별 투명도

| 승인 상태 | 표시 방식 |
| --- | --- |
| 승인 대기 | 반투명 |
| 승인 완료 | 불투명 |
| 반려 | 빨간색 테두리 |

---

### 5-3. 휴재 기간별 색상 구분

| 휴재 기간 | 색상 |
| --- | --- |
| 단기 휴재 (1-3일) | 연한 회색 |
| 중기 휴재 (4-14일) | 회색 |
| 장기 휴재 (15일 이상) | 진한 회색 + 경고 아이콘 |

---
## 🎨 UI/UX 개선 포인트

- 색상 및 패턴으로 근태 유형 구분
- 휴재 일수 표시 (예: "45일째")
- 장기 휴재 경고 표시
