## 🎯 목표

### 핵심 목표
- 일일 건강 상태 추적으로 조기 위험 감지
- 연속 출근 패턴 분석 및 건강 예측
- 주간/월간 건강 리포트 자동화

---

## 📋 상세 요구사항

### 1-1. 데일리 체크인 시 건강 설문

#### 현재 상황
- 주간/월간 건강 설문만 존재

#### 개선 사항
- 매일 체크인 시 간단한 건강 설문 자동 표시

#### 설문 항목
| 항목 | 측정 방식 |
| --- | --- |
| 손목 통증 정도 | 1-10점 |
| 수면 시간 | 전날 밤 시간 입력 |
| 피로도 | 1-10점 |
| 오늘 작업 의욕 | 1-10점 |

---

### 1-2. 연속 출근 건강 수치 예측

#### 기능 설명
| 조건 | 시스템 동작 |
| --- | --- |
| 연속 출근 7일 이상 | 건강 점수 자동 감소 경고 |
| 연속 출근 14일 이상 | 휴재 권장 알림 발송 |

#### 추가 기능
- 연속 출근 패턴 분석 및 개인별 최적 근무 패턴 제안
- 과거 건강 설문 데이터 기반 머신러닝 예측 모델

---

### 1-3. 주간/월간 건강 설문 개선

| 설문 유형 | 개선 사항 |
| --- | --- |
| 주간 설문 | 매주 월요일 자동 리마인더 |
| 월간 설문 | 월말 종합 건강 리포트 자동 생성 |
| 공통 | 설문 미제출 시 알림 발송 |

---

## 🎨 UI/UX 개선 포인트

- 체크인 시 간단한 건강 설문 팝업
- 연속 출근 경고 표시
- 건강 점수 예측 그래프

---

## 📝 변경 사항

- 데일리 건강 설문에 손목 통증(1-10점), 수면 시간, 피로도(1-10점), 작업 의욕(1-10점) 항목 추가
- 설문 소요 시간 30초 → 1분으로 변경
- 연속 출근 현황 표시 및 7일 이상 시 경고 기능 추가
- 연속 출근 7일/10일/14일 시 건강 점수 예측 기능 추가 (ML 기반)
- 연속 14일 출근 시 휴재 권장 알림 발송 기능 추가
- 개인별 최적 근무 패턴 제안 기능 추가
- 주간 설문 매주 월요일 자동 리마인더 추가
- 월간 설문 제출 시 종합 건강 리포트 자동 생성 기능 추가
- 설문 미제출 시 단계별 알림 발송 (1차~3차) 추가
- 관리자 대시보드에 연속출근 7+, 연속출근 14+, 설문미제출 현황 카드 추가
- 관리자 긴급 알림에 연속 14일 출근 휴재 권장, 주간 설문 미제출 항목 추가
- API 추가: `GET /attendance/consecutive-days`, `GET /health/prediction`, `GET /health/optimal-pattern`
- API 추가: `GET/POST /surveys/weekly`, `GET/POST /surveys/monthly`, `GET /surveys/status`, `GET /reports/monthly-summary`
- API 추가: `GET /admin/consecutive-alerts`, `GET /admin/surveys/pending`
