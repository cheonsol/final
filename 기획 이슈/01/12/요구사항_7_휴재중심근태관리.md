
## 🎯 목표


### 핵심 목표
- 연차 중심 → 휴재 중심 전환
- 휴재 유형 세분화
- 야근 관리 자동화

---

## 📋 상세 요구사항

### 7-1. 연차 중심 → 휴재 중심 전환

#### 개념 변경
| 구분 | 기존 | 개선 |
| --- | --- | --- |
| 관리 방식 | 연차(휴가) 중심의 근태 관리 | 마감 중심의 휴식 근태 관리 |
| 휴식 기준 | 연차 일수 기준 | 연재 일정에 맞춘 휴식 관리 |
| 휴식 허용 | 연차 신청 필요 | 마감일에 지장 없는 휴식 허용 |
| 관점 | 일반 직원 관점 | 작가/프리랜서 관점의 유연한 근태 관리 |

---

### 7-2. 휴재 유형 세분화

| 휴재 유형 | 설명 | 비고 |
| --- | --- | --- |
| 정기 휴재 | 예정된 휴재 (명절 등) | |
| 건강 휴재 | 손목 통증 등 건강 문제 | |
| 긴급 휴재 | 갑작스러운 사정 | |
| 시즌 휴재 | 시즌 종료 후 장기 휴재 | |
| 일상 휴식 | 너무 힘든 날의 짧은 휴식 | 연재 지장 없음, 마감 내역 지정 필요, 자동 승인 가능 (설정 시) |

---

### 7-3. 야근 관리 개선

#### 자동 퇴근 시스템
| 기능 | 설명 |
| --- | --- |
| 자동 퇴근 조건 | 야근 신청을 하지 않은 경우 11시간 경과 시 자동 퇴근 처리 |
| 화면 표시 | "자동 퇴근 처리되었습니다 (11시간 경과)" |
| 수정 기능 | 실제 퇴근 시간 수정 가능 |

#### 목적
- 과로 방지
- 근무 시간 정확한 기록
