## 🎯 목표

### 사용자 스토리
작가로서, 로그인 시 체크인을 쉽게 하고 퇴근 처리가 자동화되도록 한다.

### 핵심 목표
- 로그인 시 체크인 프로세스 개선
- 자동 퇴근 시스템으로 과로 방지
- 근무 시간 정확한 기록

---

## 📋 상세 요구사항

### 2-1. 로그인 시 체크인 안내

#### 현재 상황
- 로그인 시 자동 체크인

#### 개선 사항
| 단계 | 시스템 동작 |
| --- | --- |
| 1 | 로그인 시 오늘 날짜 기준 체크인 여부 확인 |
| 2 | 체크인 안되어 있는 경우 출근 여부 확인 팝업 표시 |
| 3 | "오늘 출근하시나요?" → 예/아니오 선택 |
| 4-1 | 예 선택 시 체크인 처리 |
| 4-2 | 아니오 선택 시 휴재/휴가 신청 안내 |

---

### 2-2. 로그아웃 미처리 관리

#### 문제점
- 로그아웃을 안한 경우 자동 처리 필요

#### 개선 사항
| 기능 | 설명 |
| --- | --- |
| 자동 퇴근 시간 설정 | 11시간 경과 시 자동 퇴근 |
| 자동 퇴근 알림 | 자동 퇴근 시 알림 발송 및 확인 요청 |
| 수동 확인 제출 양식 | 실제 퇴근 시간 입력, 미체크인 사유 입력, 관리자 확인 요청 |
