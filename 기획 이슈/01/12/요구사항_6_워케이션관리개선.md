## 📋 상세 요구사항

### 6-1. 시작/종료 보고서 제출

#### 워케이션 시작 시: 시작 보고서 제출 폼
| 항목 | 설명 |
| --- | --- |
| 목적 재확인 | 워케이션 목적 확인 |
| 예상 작업 계획 | 워케이션 기간 중 작업 계획 |
| 연락처 확인 | 비상 연락처 확인 |

#### 워케이션 종료 시: 종료 보고서 제출 폼
| 항목 | 설명 |
| --- | --- |
| 완료한 작업 요약 | 워케이션 기간 중 완료한 작업 |
| 작업량 | 완성 화수 등 정량적 성과 |
| 다음 연재 준비도 | 다음 연재 준비 상태 |
| 개선 사항 및 피드백 | 워케이션 관련 피드백 |

### 6-2. 팀 단위 Todo 리스트 및 일정 분담

#### 기능 설명
| 기능 | 설명 |
| --- | --- |
| 팀 단위 Todo 리스트 생성 | 팀 전체 할 일 목록 생성 |
| 일정 분담 체크리스트 | 각 팀원별 담당 작업 할당 |
| 작업 완료 시 체크 | 완료된 작업 체크 |
| 진행률 자동 계산 | 50%, 67% 등 자동 계산 |
| 차트로 시각화 표시 | 진행률 차트 표시 |

#### 목적
- 체크인/보고서만이 아닌 실제 업무 진행 상황 관리
- 팀 전체 작업 현황 한눈에 파악
