
## 🎯 목표

### 사용자 스토리
에이전시 관리자로서, 팀별 휴재 현황과 장기 휴재 작품을 한눈에 파악하고 싶습니다.

### 핵심 목표
- 팀(작품)별 관리 캘린더 강화
- 휴재 현황 종합 표시
- 장기 휴재 경고 시스템

## 📋 상세 요구사항

### 8-1. 팀(작품)별 관리 캘린더 강화

#### 핵심 정보 표시 - 이번 달 팀별 휴재 발생 현황
| 표시 항목 | 설명 |
| --- | --- |
| 팀별 휴재 건수 | 팀별 휴재 발생 건수 |
| 휴재 유형 분포 | 휴재 유형별 분포 현황 |
| 전월 대비 증감률 | 전월 대비 휴재 증감률 |

#### 어떤 작품에서 휴재가 집중되는지
| 표시 항목 | 설명 |
| --- | --- |
| 작품별 휴재 빈도 | 작품별 휴재 발생 빈도 |
| 휴재 집중 기간 하이라이트 | 휴재가 집중되는 기간 강조 표시 |

---

### 8-2. 기간/시즌 휴재 중인 작품 현황

| 표시 항목 | 설명 | 예시 |
| --- | --- | --- |
| 휴재 일수 표시 | 몇 일째 휴재 중인지 명확히 표시 | "시즌 휴재 45일째" |
| 장기 휴재 경고 표시 | 장기 휴재 작품 경고 아이콘 | |

#### 휴재 기간별 색상 구분
| 휴재 기간 | 색상 |
| --- | --- |
| 단기 휴재 (1-3일) | 연한 회색 |
| 중기 휴재 (4-14일) | 회색 |
| 장기 휴재 (15일 이상) | 진한 회색 + 경고 아이콘 |
