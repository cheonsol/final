

## 🎯 목표

### 사용자 스토리
관리자로서, 담당자별로 적절한 권한을 부여하고 싶습니다.

### 핵심 목표
- 담당자별 권한 세분화
- 권한 기반 자동 필터링
- 보안 강화

---

## 📋 상세 요구사항

### 9-1. 담당자별 권한 부여

#### 현재 상황
- 에이전시/팀 선택만 가능

#### 개선 사항 - 담당자별 권한 설정
| 권한 등급 | 조회 범위 |
| --- | --- |
| 편집자 | 담당 작품/팀만 조회 가능 |
| 팀장 | 소속 팀 전체 조회 |
| 관리자 | 전체 에이전시 조회 |

---

### 9-2. 권한 기반 자동 필터링

| 기능 | 설명 |
| --- | --- |
| 캘린더 필터 자동 적용 | 권한에 따른 캘린더 필터 자동 적용 |
| 목록 제외 | 권한 없는 작품/팀은 목록에서 제외 |
