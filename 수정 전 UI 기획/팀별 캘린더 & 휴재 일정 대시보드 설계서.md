- 팀별 캘린더 & 휴재 일정 대시보드 설계서 -
웹 작가 에이전시 HR 시스템 (설계 단계 · UI 포함)

────────────────────────────────

0️ 문서 목적 및 범위

목적
웹 작가 에이전시에서
팀·작품 단위로 연재 및 휴재 현황을 한눈에 파악할 수 있는
캘린더 기반 운영 대시보드를 설계한다.

이 대시보드는
‘개인 일정 관리’가 아니라
‘에이전시 운영 상태를 모니터링’하는 것이 목적이다.

범위
- 팀/에이전시 단위 캘린더 구조
- 휴재 일정 중심 이벤트 모델
- 운영자 관점의 현황 요약 + 일정 흐름
- UI 구상 포함 (구현 아님)
- 캘린더 구현을 위한 라이브러리·기술 스택 고려

범위 제외
- 개인 근태 상세 플로우
- 건강 설문 상세 로직
- 워케이션 관리 전체 플로우

────────────────────────────────

1️ 캘린더의 성격 정의

이 캘린더는
개인의 일정을 기록하는 도구가 아니라
에이전시 전체 운영 상태를 한눈에 보여주는 관리 대시보드이다.

운영자가 캘린더를 보며 즉시 파악해야 할 핵심 정보
- 이번 달 팀별 휴재 발생 현황
- 어떤 작품에서 휴재가 집중되는지
- 휴재 유형 분포(정기/건강/긴급/시즌)
- 휴재로 인해 연재 흐름이 끊기는 구간

────────────────────────────────

2️ 캘린더 핵심 표시 대상

중심 이벤트
- 작품별 연재 일정
- 작품별 휴재 일정

보조 정보
- 휴재와 겹치는 경우에만 표시되는 마감 경고

캘린더는
사람(개인)이 아니라
팀 · 작품 중심으로 구성한다.

────────────────────────────────

3️ UI 구성 개념 (대시보드 중심)

상단 영역
- 에이전시 / 팀 선택
- 기간 선택 (월 단위 기본)

중앙 영역
- 팀 / 작품 기준 캘린더

보조 영역(요약 패널)
- 팀별 휴재 건수
- 휴재 유형 분포
- 휴재 집중 기간 표시
- 전월 대비 증감 요약

────────────────────────────────

4️ 캘린더 표시 규칙 (무엇을 어떻게 보여주는가)

4-1. 연재 일정 표시
- 작품별 정규 연재 요일을 기본 블록으로 표시
- 작품명 및 플랫폼 정보 간단 표기
- 캘린더의 기본 기준선 역할

4-2. 휴재 일정 표시 (가장 중요)
- 승인 완료된 휴재 기간을 연재 블록 위에 덮어 표시
- 회색 또는 패턴 블록으로 시각적 구분
- 휴재 유형(정기/건강/긴급/시즌)을 아이콘·태그로 표시

핵심 원칙
‘연재가 없는 날’이 아니라
‘의도적으로 휴재 중인 날’로 인식되게 한다.

4-3. 승인 상태에 따른 표시 차이
- 승인 대기 휴재: 반투명 + 승인 대기 태그
- 승인 완료 휴재: 불투명 블록 + 집계 대상 포함

────────────────────────────────

5️ 강조 및 경고 표시 규칙

5-1. 마감 경고 표시
- 휴재 기간과 마감일이 겹치는 경우에만 표시
- 해당 날짜 또는 작품 행에 경고 아이콘 부착

5-2. 휴재 집중 구간 강조
- 특정 기간 내 휴재 건수가 기준 초과 시
- 해당 날짜 영역에 배경 강조 및 라벨 표시

목적
운영자가 캘린더를 훑는 것만으로도
리스크 구간을 즉시 인지할 수 있도록 한다.

────────────────────────────────

6️ 휴재 중심 비즈니스 로직

연재 일정
- 작품 등록 시 연재 요일 설정
- 시스템이 월 단위 연재 일정 자동 생성
- 연재 일정은 삭제되지 않으며 기본값으로 유지된다

휴재 처리 원칙
- 모든 일정 변경은 ‘휴재’ 이벤트로 표현
- 연재 일정 자체는 유지하고 상태로 제어한다

휴재 신청 → 승인 → 반영 흐름
1. 휴재 신청 시 작품, 기간, 휴재 유형 입력
2. 승인 완료 시 휴재 기간 내 연재 일정 조회
3. 해당 연재 일정 상태를 CANCELED 처리
4. 동일 기간에 휴재 이벤트 생성
5. 휴재 이력을 작품 단위로 누적 기록

결과
연재 공백이 아닌
의도된 운영 결정으로서의 휴재가 명확히 표현된다.

────────────────────────────────

7️ 팀/에이전시 요약 패널 로직

요약 패널은
캘린더 데이터를 기반으로 자동 집계된다.

표시 정보
- 팀별 휴재 건수 (선택 기간 기준)
- 휴재 유형 분포
- 휴재 집중 기간
- 전월 대비 휴재 증감

캘린더와 동일한 원본 데이터를 사용하여
정합성을 유지한다.

────────────────────────────────

8️ 권한별 캘린더 관점

- 작가: 본인 작품 일정만 조회
- 편집자: 담당 팀 및 작품 일정 조회
- 관리자: 전체 에이전시 캘린더 조회

같은 캘린더라도
조회 범위는 역할에 따라 다르다.
