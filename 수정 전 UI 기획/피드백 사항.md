피드백 정리 /// 건강 관리 기능 개선

1-1. 건강 설문 시스템 강화

데일리 체크인 시 건강 설문 현재: 주간/월간 건강 설문만 존재 개선: 매일 체크인 시 간단한 건강 설문 자동 표시 손목 통증 정도 (1-10점) 수면 시간 (전날 밤) 피로도 (1-10점) 오늘 작업 의욕 (1-10점) 목적: 일일 건강 상태 추적으로 조기 위험 감지

연속 출근 건강 수치 예측 기능 추가: 연속 출근 일수에 따른 건강 수치 예측 연속 출근 7일 이상 → 건강 점수 자동 감소 경고 연속 출근 14일 이상 → 휴재 권장 알림 연속 출근 패턴 분석 및 개인별 최적 근무 패턴 제안 데이터 활용: 과거 건강 설문 데이터 기반 머신러닝 예측 모델

주간/월간 건강 설문 개선 현재: 건강 설문 기능 존재 개선: 주간 설문: 매주 월요일 자동 리마인더 월간 설문: 월말 종합 건강 리포트 자동 생성 설문 미제출 시 알림 발송

1-2. 에이전시 차원 건강 관리 필요성 검토

검토 항목 에이전시 전체 건강 지표 대시보드 필요성 팀별 건강 비교 분석 기능 건강 위험 작가 자동 식별 시스템 건강 기반 인력 배정 최적화 건강 개선 프로그램 추천 시스템

권장 사항 1단계: 개인 건강 관리 기능 완성 2단계: 팀 단위 건강 통계 제공 3단계: 에이전시 전체 건강 관리 대시보드 (필요 시)

작가 및 직원 관점 스케줄/근무 관리 개선

2-1. 작가 루틴 개선

로그인 시 체크인 안내 현재: 로그인 시 자동 체크인 개선: 로그인 시 오늘 날짜 기준 체크인 여부 확인 체크인 안되어 있는 경우 출근 여부 확인 팝업 표시 "오늘 출근하시나요?" → 예/아니오 선택 예 선택 시 체크인 처리, 아니오 선택 시 휴재/휴가 신청 안내

로그아웃 미처리 관리 문제: 로그아웃을 안한 경우 자동 처리 필요 개선: 자동 퇴근 시간 설정 (예: 11시간 경과 시 자동 퇴근) 자동 퇴근 시 알림 발송 및 확인 요청 수동 확인 제출 양식 제공 실제 퇴근 시간 입력 미체크인 사유 입력 관리자 확인 요청

2-2. 작품 팀 연재 일정 설정

연재 주기 설정 기능 추가 기능: 작품별 연재 주기 설정 7일 주기 (주 1회) 10일 주기 14일 주기 (격주) 커스텀 주기 설정 활용: 마감일 자동 계산 휴재 영향도 자동 분석 연재 일정 캘린더 자동 생성

2-3. 직원 대시보드 개선

대시보드 구성 변경 현재: 오늘 연재 일정 중심 개선: 일정/스케줄표/Todo 리스트 형태로 변경 오늘 할 일 목록 마감일 D-day 표시 진행 중인 작업 현황 다음 연재일까지 일정 상태별 관리 워케이션 중: 원격 근무 현황, 체크인/체크아웃 상태 일반 근무 중: 오늘의 작업 목록, 마감일 현황 내용 관리 개선 승인 대기보다는 작업 현황, 마감일, 진행률 중심 표시 빠른 액션: 사용자 커스터마이징 가능 자주 사용하는 기능을 빠른 액션에 추가 드래그 앤 드롭으로 순서 변경

2-4. 캘린더 시각화 개선

색상 구분 강화 현재: 선으로만 표시 개선: 근태 유형별 색상 구분 연차: 파란색 휴재: 회색/패턴 워케이션: 보라색/그라데이션 일반 근무: 초록색 야근: 주황색 상태별 투명도 승인 대기: 반투명 승인 완료: 불투명 반려: 빨간색 테두리

2-5. 워케이션 관리 개선

시작/종료 보고서 제출 기능 추가: 워케이션 시작 시: 시작 보고서 제출 폼 목적 재확인 예상 작업 계획 연락처 확인 워케이션 종료 시: 종료 보고서 제출 폼 완료한 작업 요약 작업량 (완성 화수 등) 다음 연재 준비도 개선 사항 및 피드백

팀 단위 Todo 리스트 및 일정 분담 기능 추가: 팀 단위 Todo 리스트 생성 일정 분담 체크리스트 각 팀원별 담당 작업 할당 작업 완료 시 체크 진행률 자동 계산 (50%, 67% 등) 차트로 시각화 표시 목적: 체크인/보고서만이 아닌 실제 업무 진행 상황 관리 팀 전체 작업 현황 한눈에 파악

휴재 관점의 근태 관리 시스템 재정립

3-1. 연차 중심 → 휴재 중심 전환

개념 변경 기존: 연차(휴가) 중심의 근태 관리 개선: 마감 중심의 휴식 근태 관리 연재 일정에 맞춘 휴식 관리 마감일에 지장 없는 휴식 허용 작가/프리랜서 관점의 유연한 근태 관리

휴재 유형 세분화 정기 휴재: 예정된 휴재 (명절 등) 건강 휴재: 손목 통증 등 건강 문제 긴급 휴재: 갑작스러운 사정 시즌 휴재: 시즌 종료 후 장기 휴재 일상 휴식: 너무 힘든 날의 짧은 휴식 (연재 지장 없음) 마감 내역 지정 필요 자동 승인 가능 (설정 시)

3-2. 야근 관리 개선

자동 퇴근 시스템 기능: 야근 신청을 하지 않은 경우 11시간 경과 시 자동 퇴근 처리 자동 퇴근 시 화면에 명확히 표시 "자동 퇴근 처리되었습니다 (11시간 경과)" 실제 퇴근 시간 수정 가능 목적: 과로 방지 근무 시간 정확한 기록

에이전시 관점 캘린더 대시보드 개선

4-1. 팀(작품)별 관리 캘린더 강화

핵심 정보 표시 이번 달 팀별 휴재 발생 현황 팀별 휴재 건수 휴재 유형 분포 전월 대비 증감률 어떤 작품에서 휴재가 집중되는지 작품별 휴재 빈도 휴재 집중 기간 하이라이트 기간/시즌 휴재 중인 작품 현황 몇 일째 휴재 중인지 명확히 표시 예: "시즌 휴재 45일째" 장기 휴재 경고 표시 휴재 기간별 색상 구분 단기 휴재 (1-3일): 연한 회색 중기 휴재 (4-14일): 회색 장기 휴재 (15일 이상): 진한 회색 + 경고 아이콘

4-2. 권한 관리 개선

담당자별 권한 부여 현재: 에이전시/팀 선택만 가능 개선: 담당자별 권한 설정 편집자: 담당 작품/팀만 조회 가능 팀장: 소속 팀 전체 조회 관리자: 전체 에이전시 조회 권한에 따른 캘린더 필터 자동 적용 권한 없는 작품/팀은 목록에서 제외

4-3. 기간/시즌 휴재 관리 로직

장기 휴재 시 팀 운영 관리 문제: 장기 휴재 중인 작품 팀의 업무 처리 방안 불명확 개선: 휴재 중 업무 관리 시스템 휴재 기간 중 팀원별 업무 배정 대체 작업 계획 수립 휴재 복귀 준비 일정 관리 휴재 기간 중 진행률 추적 휴재 관리 대시보드 휴재 중인 작품 목록 휴재 일수 표시 휴재 중 진행 중인 작업 현황 복귀 예정일 및 준비도

4-4. 요약 패널 개선

기간/시즌 휴재 전용 패널 기존: 팀별 휴재 건수만 표시 개선: 장기 휴재 작품 전용 섹션 시즌 휴재 중인 작품 목록 휴재 일수 및 복귀 예정일 휴재 중 업무 처리 현황 휴재 관리 상태 (정상/주의/위험) 휴재 관리 액션 휴재 연장 신청 관리 복귀 일정 조정 휴재 중 업무 재배정
