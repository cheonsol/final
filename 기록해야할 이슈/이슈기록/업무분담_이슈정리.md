## 🏛️ 에이전시 신규 페이지 (권한 체계)

```
에이전시 권한 구조
├── 에이전시 회사
├── 담당담당자 (담당자들을 담당)
└── 담당자
```

| 담당 | 페이지 | 기능 |
|------|--------|------|
| 진혁 | 인원 목록 | 담당자/소속 아티스트/어시스턴트/편집자 목록 |
| 남규 | 등록/권한 | 등록/탈퇴 화면, 담당자 권한 부여(직급/작가), 편집자 지정 |
| 혜정 | 관리/승인 | 작품 조회, 전체 팀 관리, 건강상태 확인, 휴가 승인/반려 |
