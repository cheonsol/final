## 🎨 작가 대시보드

| 이슈 | 결정사항 |
|------|----------|
| 상태별(연재 중/장기 휴재/워케이션) 화면 구성 재정립 필요 | 상태에 맞는 정보 및 스케줄 관리 화면 재설계 |
| 건강 설문 UI 디자인 부재 (체크인 시 간단 설문) | 디자인 필요 |
| 진행중인 에피소드 관련 내용 없음 | 아이디어 필요 |
| '빠른 작업' 기능 정의 불명확 (커스텀 리모컨 vs 일정 관리) | 명확화 필요 |
| 시각적 표현 부족 (차트/그래프/태스크 그래프 활용 제안) | 시각화 추가 검토 |
| 건강 현황 작가 노출 여부 | 유지/삭제 결정 필요 |
| 작업 현황 - 생산성 기준 불명확, 완성 컷 개념 추상적 | 대시보드 재설계 예정, 체크인(출석)은 필수 유지 |
